Machine learning based cyberattack detection

Created a cyberattack detection model by applying unsupervised machine learning (iForest/LOF) on network traffic data, generated data analysis with Python/Splunk. Detected ICMP-DDoS attack from 79 malicious IPs and 101 victims. Moreover, adversarial samples were generated to trick a Logistic Regression model to avoid detection.
